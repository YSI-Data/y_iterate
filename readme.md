# y_iterate

[![sampctl](https://shields.southcla.ws/badge/sampctl-y_iterate-2f2f2f.svg?style=for-the-badge)](https://github.com/YSI-Data/y_iterate)

## Installation

To install just this one library:

```bash
sampctl package install YSI-Data/y_iterate
```

Include in your code and begin using the library:

```pawn
#include <YSI-Data/y_iterate>
```
